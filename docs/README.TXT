NifUtils
========

Copyright (c) 2012, NifUtils.
All rights reserved.


NifUtils are a collection of tools supporting NetImmerse files (.nif) such as creating from 
older file versions, adding collision information to existing meshes, decoding MoppData of 
bhkMoppBvTreeShape and visualize meshes encoded in Chunks section of bhkCompressMeshData.
